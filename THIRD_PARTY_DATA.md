# Third-Party Data

Q2DM uses records associated with 2D materials databases, including 2DMatPedia-related material records and derived local training caches.

The repository contains code, documentation, and representative result artifacts. It does not contain the full third-party raw datasets, processed dataset caches, or scientific paper PDFs.

## Dataset-Derived Paths

Referenced dataset-derived paths:

```text
ModelB/data/raw/2dmatpedia_full.json
ModelB/data/processed/dataset.npz
ModelC/data/raw/2dmatpedia_full.json
ModelA/data/mo_dataset_cache.npz
```

These paths are excluded from version control.

## Citation Boundary

Dataset records, upstream database content, and scientific papers carry citation and redistribution obligations separate from this repository's MIT license. The bibliography in `docs/bibliography.md` records the main papers that informed the project.
