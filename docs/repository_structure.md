# Repository Map

## Overview

This repository is organized around the retained Q2DM model lines, shared documentation, and selected result artifacts.

## Top-Level Layout

- `ModelA/` - intermediate graph-QNN line with GCN-style components, angle encoding, tests, and representative artifacts.
- `ModelB/` - primary Graph-QNN pipeline, including core source, tests, retained artifacts, experimental VQC code, and v4 design notes.
- `ModelC/` - Coulomb-matrix/amplitude-encoding baseline retained for comparison.
- `docs/` - architecture, data provenance, dependency, project-history, and technical documentation.
- `references/` - bibliography pointers.
- `THIRD_PARTY_DATA.md` - dataset and attribution boundary.
- `NOTICE` - third-party provenance and licensing notice.

## Omitted Artifact Classes

- Full raw dataset dumps and processed dataset caches.
- Stored copies of scientific paper PDFs.
- Trained checkpoint binaries.
- Complete experiment directory trees.

## Limitations

This repository is a project showcase and technical artifact index, not a complete experiment archive. Dataset-derived files remain outside version control.
